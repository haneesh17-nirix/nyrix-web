# nyrix-web

Product landing page for **Nyrix: BME Tracker** — hosted at [www.aazhara.in](https://www.aazhara.in).

The actual application lives at [nyrix.aazhara.in](https://nyrix.aazhara.in).

## Hosting

Deployed via **GitHub Pages** from the `main` branch root.

## Local Preview

Just open `index.html` in a browser — no build step required.

## DNS Setup (Wix DNS for aazhara.in)

| Type | Name | Value |
|---|---|---|
| CNAME | `www` | `haneesh17-nirix.github.io` |

After adding, GitHub Pages will auto-provision SSL via Let's Encrypt.
